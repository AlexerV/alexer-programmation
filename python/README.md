---

## 🧰 Prérequis

- Installer [Python 3.x](https://www.python.org/downloads/) sur ta machine.

### 📦 Vérifier l’installation
Selon votre système :

Windows :
```bash
python --version
```

Linux/macOS :
```bash
python3 --vserion
```
> ⚠️ Sur Linux/macOS, python pointe parfois vers Python 2.7. Utilisez donc python3 si nécessaire.
